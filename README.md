## 分析约束速查

| 检查 | 阈值 | 参考 |
|------|------|------|
| VIF | >10 剔除 | `快速速查表.md` |
| 过离散 | >1.5 换负二项 | `I-统计建模/` |
| \|r\| | >0.7 删变量 | `快速速查表.md` |
| PERMANOVA | 先做 betadisper | `C-多元统计/` |
| 空间自相关 | 检验 Moran's I | `D-空间生态学/` |
| 样本量 | 每参数≥10 | `I-统计建模/` |
