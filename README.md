## 分析约束速查

> 📖 以下阈值和规则均出自 *Numerical Ecology* (Legendre & Legendre 2012) 及配套 R 包文档

| 检查 | 阈值 | 参考 |
|------|------|------|
| VIF | >10 剔除 | 书第5章 / `快速速查表.md` |
| 过离散 | >1.5 换负二项 | 书第3章 / `I-统计建模/` |
| \|r\| | >0.7 删变量 | 书第2章 / `快速速查表.md` |
| PERMANOVA | 先做 betadisper | Anderson (2001) / `C-多元统计/` |
| 空间自相关 | 检验 Moran's I | 书第13章 / `D-空间生态学/` |
| 样本量 | 每参数≥10 | 书第1章 / `I-统计建模/` |
