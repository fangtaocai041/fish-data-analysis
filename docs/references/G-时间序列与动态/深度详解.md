## 一、ARIMA vs GAM：何时用什么？

| | ARIMA | GAM |
|:--|:------|:----|
| **趋势** | 需要差分去除 | 自动拟合平滑趋势 |
| **季节性** | 需要手动指定周期 | 需要手动设置循环平滑 |
| **预测** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ (外推危险) |
| **解释性** | ⭐⭐ | ⭐⭐⭐⭐⭐ |
| **残差自相关** | 自动建模 | 需要手动检查 |
| **适用 n** | n ≥ 50 | n ≥ 30 |

## 二、早期预警信号 (EWS)：不能证明因果

```
临界减慢 (CSD) → 系统接近临界点时：
1. 恢复速率变慢 → 方差增大
2. 自相关增强 (AR1 ↑)
3. 偏度可能改变

三个"不能"：
❌ EWS 不能预测"何时"发生临界转换
❌ EWS 不能告诉你"哪个驱动因素"导致转换
❌ EWS 显著 ≠ 临界转换一定会发生（可能有假阳性）

稳妥做法：
→ 至少报告 3 个 EWS 指标（方差+AR1+偏度）
→ 如果三个指标一致 → 信号更可信
→ 如果不一致 → 讨论为什么某些指标不响应
```

## 三、状态空间模型：分离观测误差和过程误差

```
为什么生态时间序列需要状态空间模型？
→ 我们的观测 = 真实状态 + 观测误差
→ 普通回归：观测误差和过程误差混在一起 → 高估变异性
→ 状态空间模型：显式分离两者

R 包推荐：MARSS（多变量自回归状态空间）
```
