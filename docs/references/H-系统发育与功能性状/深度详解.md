# 🌳 H — 系统发育与功能性状 · 深度详解

> 物种不是独立的——它们共享进化历史。忽视系统发育 → 假阳性；正确整合 → 揭示进化-生态耦合。

## 一、系统发育信号：Pagel's λ 和 Blomberg's K

### λ 和 K 告诉你不同的事

```
λ ≈ 1, K ≈ 1 → 性状进化符合布朗运动（近缘种相似）
λ ≈ 0, K < 1 → 无系统发育信号（性状独立于系统发育）
K > 1      → 近缘种比布朗运动预期更相似（趋异不足）

两者都报告，因为：
→ λ 告诉"信号强度"，K 告诉"相对于BM的强度"
→ 有时 λ 显著但 K 不显著 → 需要讨论原因
```

## 二、PGLS：使用前必须检验信号

```
❌ 错误做法：
  → 直接跑 PGLS，不管有没有系统发育信号
  → 如果 lme4 和 PGLS 结果不同，选 p 值小的那个

✅ 正确做法：
  1. 先检验系统发育信号（λ + K）
  2. 如果有信号 → PGLS
  3. 如果没有信号 → 普通 OLS/GLM 即可
  4. 让 PGLS 同时估计 λ（不固定）
  5. 报告："λML = 0.76, p < 0.01，使用 PGLS 校正系统发育"
```

## 三、系统发育群落生态学：NRI/NTI 解读

```
NRI > 0 (ses.MPD < 0) → 系统发育聚集
  → 共存物种比随机预期更近缘
  → 可能原因：环境筛选（只允许特定分支的物种生存）

NRI < 0 (ses.MPD > 0) → 系统发育发散
  → 共存物种比随机预期更远缘
  → 可能原因：竞争排斥（近缘种不能共存）

NTI > 0 → 终端聚集（最近亲缘种尺度）
NTI < 0 → 终端发散

NRI 和 NTI 的四种组合反映了不同生态过程
```
