## A — 数据预处理

### A1 异常值检测

#### Boxplot 规则 (Q1-1.5IQR)

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 单变量、近似正态分布的连续数据；快速初筛 |
| **不适用场景** | ① 严重偏态分布（会标记过多正常点）② 小样本 (n<10) ③ 多变量异常值检测 |
| **数据最低要求** | n ≥ 20 才比较可靠 |
| **稳健性** | ⭐⭐⭐ 对偏态敏感；极端偏态下建议用 adjusted boxplot (medcouple) |
| **假设违反后果** | 偏态数据：可能将 5-10% 的正常数据标记为异常值 |
| **替代方案** | 偏态 → adjusted boxplot；多变量 → Mahalanobis 距离 / Isolation Forest |

#### Cook's Distance

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 回归模型诊断，识别同时影响拟合值和回归系数的强影响点 |
| **不适用场景** | ① 非回归场景 ② 共线性严重的模型（一个异常值可能"掩护"另一个）③ 非线性模型（需改用其他方法） |
| **数据最低要求** | 至少需要 n > 2k（k为参数个数）；推荐 n/k > 10 |
| **稳健性** | ⭐⭐⭐⭐ 对回归诊断非常有效；阈值 D > 4/n 或 1 是经验规则 |
| **假设违反后果** | 共线性高时 Cook's D 可能低估某些异常点的影响 |
| **替代方案** | DFFITS, DFBETAS（分别看拟合值和系数影响） |

#### Mahalanobis 距离

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 多变量异常值检测（考虑变量间协方差结构） |
| **不适用场景** | ① 变量数 > 样本数（协方差矩阵不可逆）② 严重非正态数据 ③ 分类变量 |
| **数据最低要求** | n > 3p（p=变量数）；可靠估计需要 n > 5p |
| **稳健性** | ⭐⭐⭐ 对非正态有一定容忍度，但严重偏态需要稳健版本 |
| **假设违反后果** | 非正态：可能漏掉真正的异常值或标记正常值 |
| **替代方案** | 稳健 Mahalanobis (MCD估计)；变量数多 → PCA + 前几个主成分的 Mahalanobis |

### A2 正态性检验与转换

#### Shapiro-Wilk 检验

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 小到中等样本 (n < 2000-5000) 的正态性检验 |
| **不适用场景** | ① n > 5000（几乎总是显著）② 只需要评估偏离程度而非是否显著 |
| **数据最低要求** | n ≥ 3 |
| **稳健性** | ⭐⭐⭐⭐⭐ 在常用正态性检验中统计效力最高 |
| **假设违反后果** | n 很大时：即使是生物学上无关紧要的微小偏离也会显著 → 过度拒绝正态性假设 |
| **替代方案** | n > 5000 → Q-Q 图 + 偏度/峰度；结构性的非正态 → 直接使用非参数方法 |

#### Box-Cox 转换

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 正偏态连续数据，目标是使数据更接近正态或稳定方差 |
| **不适用场景** | ① 负值或零值（λ 估计不稳定）② 已接近正态的数据 ③ 有序分类变量 |
| **数据最低要求** | n ≥ 20 才能可靠估计 λ |
| **稳健性** | ⭐⭐⭐ λ 估计对异常值敏感；先处理异常值再做 Box-Cox |
| **假设违反后果** | 有异常值时：λ 可能被拉向极端值 → 转换效果差 |
| **替代方案** | 零值 → log(x+c) 或 Yeo-Johnson；偏态不确定 → 秩转换或非参数方法 |

#### Hellinger 转换（群落数据）

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 群落数据（物种-样方矩阵），解决"双零问题"后使用欧氏距离 |
| **不适用场景** | ① 需要保留原始丰度信息的场景 ② 环境变量数据 |
| **数据最低要求** | 样方数 ≥ 10，物种数 ≥ 5 |
| **稳健性** | ⭐⭐⭐⭐⭐ 非常稳健，是群落数据标准化的首选 |
| **假设违反后果** | 正确使用无严重后果；错误用于环境数据会导致不可解释的结果 |
| **替代方案** | Chord 转换、Wisconsin 标准化；Bray-Curtis 距离则不需要转换 |

### A3 缺失值处理

#### 多重插补 (MICE)

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | MCAR 或 MAR 缺失，缺失比例 5%-50% |
| **不适用场景** | ① MNAR（缺失原因与缺失值本身有关）② 缺失 > 60% ③ 完全随机的孤立缺失 (< 3%，简单删除即可) |
| **数据最低要求** | 每个变量至少 50 个完整观测；推荐辅助变量与目标变量相关 |
| **稳健性** | ⭐⭐⭐⭐ MAR 下非常稳健；MNAR 下偏倚取决于缺失机制 |
| **假设违反后果** | MNAR 时：插补值系统偏倚，且偏倚方向不可预测 |
| **替代方案** | MNAR → 模式混合模型或选择模型；缺失太多 → 放弃该变量 |

#### KNN 插补

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 连续变量，变量间有合理的相关性结构 |
| **不适用场景** | ① 分类变量（需改用 KNN 分类模式）② 高维数据（维度灾难）③ 缺失比例高 (> 30%) |
| **数据最低要求** | n > 30，p < 15 时效果好 |
| **稳健性** | ⭐⭐⭐ 对 k 的选择敏感；k 太小→噪声大，k 太大→过度平滑 |
| **假设违反后果** | 高维时：最近的"邻居"实际很远，插补不准确 |
| **替代方案** | 高维 → 随机森林插补 (missForest)；时间序列 → 线性插值/上次观测结转 |

## B — 多样性分析

### B1 α多样性

#### Shannon-Wiener (H') vs Simpson (D)

| 维度 | Shannon | Simpson |
|:-----|:--------|:--------|
| **敏感方向** | 稀有物种 | 优势物种 |
| **适用场景** | 关注物种丰富度和稀有物种 | 关注群落均匀度和优势度 |
| **不适用场景** | 采样不充分时大幅低估 | 采样不充分时影响较小 |
| **数据最低要求** | 抽样覆盖率 ≥ 80% 才可靠 | 抽样覆盖率 ≥ 60% 即可 |
| **稳健性** | ⭐⭐⭐⭐ 对抽样完整性敏感 | ⭐⭐⭐⭐⭐ 对抽样不完整比较稳健 |
| **假设违反后果** | 采样不足：严重低估真实多样性 | 采样不足：轻微低估 |
| **选择建议** | **两者都报告**，不要只选一个 |

#### Chao1 估计器

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 估计群落中的未观测物种数（基于稀有物种的丰度信息：singleton 和 doubleton） |
| **不适用场景** | ① 只有 presence/absence 数据（用 Chao2）② 稀有物种采样严重不足（没有 singleton 就没法估计）③ 群落个体数极少 |
| **数据最低要求** | 至少捕获了 singleton 和 doubleton；样本量 > 50 个个体 |
| **稳健性** | ⭐⭐⭐ 对稀有物种的采样充分性极其敏感；Chao1 的下界估计比上界稳定 |
| **假设违反后果** | 如果 singleton 和 doubleton 都很少：Chao1 ≈ S_obs，低估真实丰富度 |
| **替代方案** | ACE（基于覆盖度的估计）；iNEXT 的稀疏外推框架（更现代） |

---

### B2 β多样性

#### Bray-Curtis 相异性

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 基于丰度的群落相异性（最流行的 β 多样性指标） |
| **不适用场景** | ① 需要满足三角不等式的距离方法（如 PCoA 的理论推导）② 关注 β 多样性组分分解时 |
| **数据最低要求** | 每样方至少有 3-5 个物种 |
| **稳健性** | ⭐⭐⭐⭐⭐ 非常稳健，是群落生态学的默认选择 |
| **假设违反后果** | 没有强假设，但对总丰度差异和组成差异混淆的情况需谨慎 |
| **替代方案** | Bray-Curtis 是通常的最佳选择；关注周转 → Baselga 分解 |

#### Baselga β 分解 (β = β_sim + β_sne)

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 区分 β 多样性的两个组分：物种周转 (turnover) 和嵌套 (nestedness) |
| **不适用场景** | ① 样方数太少 (< 5) ② 需要对不同指数进行敏感性比较时 |
| **数据最低要求** | 至少 3 个样方，每样方 ≥ 3 物种 |
| **稳健性** | ⭐⭐⭐⭐ 依赖于基础相异性指数的选择（Sørensen vs Jaccard）；两者通常给出相似结论 |
| **假设违反后果** | 选择不合适的指数可能导致对周转/嵌套的错误归因 |
| **替代方案** | Podani 分解（对稀有种更敏感）；Legendre 的 β 多样性统计框架 |

### B4 功能多样性

#### FRic (功能丰富度) — 凸包体积

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 连续功能性状，衡量群落占据的功能空间大小 |
| **不适用场景** | ① 物种数 < 性状数+1（凸包退化为平面）② 包含分类性状而未编码 ③ 缺失性状过多 |
| **数据最低要求** | 物种数 S > 性状数 T+1；推荐 S > 2T |
| **稳健性** | ⭐⭐⭐ 对异常值和极端性状极其敏感；必须在计算前标准化性状 |
| **假设违反后果** | 未标准化：凸包被大尺度变量主导；异常值：极大膨胀 FRic |
| **替代方案** | S < T+1 → FDis（功能离散度，无此限制）；异常值 → 先 winsorize |

## C — 多元统计分析

### C1 排序分析

#### PCA (主成分分析)

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 连续环境变量（标准化后），探索线性相关结构 |
| **不适用场景** | ① 群落物种数据（除非 Hellinger 转换后）② 非线性关系 ③ 分类变量为主的数据 |
| **数据最低要求** | n > p（样本 > 变量）；可靠：n > 5p |
| **稳健性** | ⭐⭐⭐ 对异常值敏感；一个极端值就可能扭曲整个排序 |
| **假设违反后果** | 群落数据：双零被解释为"相似"（实际上它们可能来自完全不同环境）；非线性：低估真实结构 |
| **替代方案** | 群落数据 → CA/DCA；非线性 → NMDS；分类数据 → MCA |

#### NMDS (非度量多维尺度分析)

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 任何相异矩阵；不需要分布假设；群落数据的最佳探索工具 |
| **不适用场景** | ① 需要精确分解方差（NMDS 无法做 VPA）② 假设检验的框架（改用 PERMANOVA） |
| **数据最低要求** | 样方数 ≥ 6（k=2时），≥ 10（k=3时） |
| **稳健性** | ⭐⭐⭐⭐ 对数据分布非常稳健；但 Stress 可能陷入局部极小值 |
| **假设违反后果** | 局部极小 → 多尝试几次 (try=100)；Stress > 0.2 → 增加维度或换方法 |
| **替代方案** | 高 Stress → 尝试不同距离指数；需要方差分解 → db-RDA |

#### RDA (冗余分析) vs CCA (典范对应分析)

| 维度 | RDA | CCA |
|:-----|:----|:-----|
| **假设** | 物种-环境线性响应 | 物种-环境单峰响应 |
| **梯度长度要求** | DCA 第一轴 < 3 SD | DCA 第一轴 > 4 SD |
| **适用场景** | 短环境梯度、同质生境 | 长环境梯度、异质生境 |
| **不适用场景** | 长梯度（选 CCA） | 短梯度（RDA 更合适） |
| **共线性容忍** | ⭐⭐⭐ VIF > 10 必须处理 | ⭐⭐⭐ VIF > 10 必须处理 |
| **稳健性** | ⭐⭐⭐⭐ 对适度违反线性假设有一定容忍 | ⭐⭐⭐ CCA 弓形效应可能制造虚假的第二轴 |
| **假设违反后果** | RDA 用于长梯度：低估物种关系 | CCA 用于短梯度：可能不稳定 |
| **替代方案** | db-RDA（距离矩阵可直接用 Bray-Curtis） | 弓形效应严重 → 改用 NMDS + envfit |

#### 方差分解 (VPA)

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 分解群落变异为不同解释变量组的贡献（如环境 vs 空间） |
| **不适用场景** | ① 变量组高度相关时 ② 负的"解释方差"出现时 |
| **数据最低要求** | 每个变量组的变量数 < 样方数/3 |
| **稳健性** | ⭐⭐⭐ 调整 R² 可能出现负值（当解释变量组"解释"少于随机噪声时） |
| **假设违反后果** | 变量组高度相关 → "共享变异"被高估 |
| **替代方案** | 变量组高度相关 → 用单一模型+部分R²而非VPA；或用随机森林的变量重要性 |

### C2 组间差异检验

#### PERMANOVA (adonis2)

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 基于距离矩阵的多变量组间差异检验（最灵活的群落差异检验） |
| **不适用场景** | ① 平衡设计 + 正态假设满足时（PERMANOVA less powerful than MANOVA）② 组内离散度不同时（显著可能是离散度差异） |
| **数据最低要求** | 每组 ≥ 4 个样方；推荐 ≥ 6 |
| **稳健性** | ⭐⭐⭐⭐ 对分布假设非常稳健；但对组内离散度差异敏感 |
| **假设违反后果** | 组内离散度不同 → 假阳性：betadisper 必须首先检验 |
| **替代方案** | 离散度不同 → 分层 PERMANOVA 或改用 ANOSIM（秩方法） |

#### betadisper（多元同质性检验）

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | PERMANOVA 的前置检验：检查各组到组质心的平均距离是否相等 |
| **不适用场景** | ① 单独作为主要分析方法 ② 组样本量极度不平衡时 |
| **数据最低要求** | 每组 ≥ 3 个样方 |
| **稳健性** | ⭐⭐⭐ 对组样本量不平衡有一定容忍度 |
| **假设违反后果** | p < 0.05 → PERMANOVA 显著可能是离散度差异而非位置差异 → 需谨慎解释 |
| **替代方案** | 如果 betadisper 显著 → 考虑多层次分析或分别分析组内离散度 |

## D — 空间生态学

### D1 空间自相关

#### Moran's I

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 检测全局空间自相关（正值=聚集，负值=分散，0=随机） |
| **不适用场景** | ① 需要空间尺度信息（改用 Variogram）② 局部异质性（改用 LISA）③ 不规则采样密度 |
| **数据最低要求** | n ≥ 30；空间权重矩阵的定义需要至少 3-5 个邻居 |
| **稳健性** | ⭐⭐⭐⭐ 经典方法，但结果对空间权重矩阵的定义非常敏感 |
| **假设违反后果** | 权重矩阵选错 → 结论可能完全相反；邻居定义过少→高估自相关 |
| **替代方案** | 多尺度 → MEM/PCNM；局部 → LISA (Local Moran's I) |

#### Mantel 检验

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 检验两个距离矩阵之间的相关性（如群落相异 vs 地理距离） |
| **不适用场景** | ① 统计效力过低（低统计效力已被大量文献批判）② 有更好的替代方案时 |
| **数据最低要求** | n ≥ 30（效果较差），推荐 n ≥ 50 |
| **稳健性** | ⭐⭐ 统计效力低！即使真实效应存在也可能不显著；对距离矩阵中的非线性关系不敏感 |
| **假设违反后果** | 假阴性率高：真实的距离衰减关系可能被漏掉 |
| **替代方案** | **强烈推荐** MEM + RDA 方差分解，或 db-RDA + 空间变量，或 Procrustes 检验 |

### D2 空间尺度分解

#### MEM / PCNM (空间特征变量)

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 从空间坐标中提取多尺度正交特征变量，用于群落-环境-空间的方差分解 |
| **不适用场景** | ① 只有几个采样点 (< 20) ② 采样点在1D（河流）上时需要 AEM 而非 MEM |
| **数据最低要求** | n ≥ 20；n ≥ 50 才能可靠区分多尺度 |
| **稳健性** | ⭐⭐⭐⭐ 非常强大的方法；正特征值的选择标准需小心（全局 vs 局部结构） |
| **假设违反后果** | 正/负特征值混用 → 负空间自相关（竞争性结构）被错误纳入 → 假阳性 |
| **替代方案** | 1D（河流）→ AEM；不规则采样 → dbMEM；小样本 → Moran's Eigenvector Maps (MEM) 的最小特征值选择 |

## E — 生态位与物种分布

### E1 MaxEnt

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 仅存在点 (presence-only) 数据；是当前最流行的物种分布模型 |
| **不适用场景** | ① 存在/不存在 (PA) 数据（GLM/GAM 更合适）② 存在点 < 15-20 ③ 严重采样偏差且未校正 |
| **数据最低要求** | ≥ 15 个存在点（bare minimum）；≥ 50 推荐；≥ 100 理想 |
| **稳健性** | ⭐⭐⭐ 默认参数不一定最优（需调 feature types + regularization multiplier）；MaxEnt 的默认设置对现代版本 (maxnet) 已大幅改善 |
| **假设违反后果** | 采样偏差 → 模型偏向采样密集区；未调参 → 过拟合；小样本 → AUC 虚高 |
| **替代方案** | PA 数据 → GLM/GAM/RF；样本少 → BIOCLIM/Domain（简单但透明）；需要不确定性 → 贝叶斯 SDM |

#### 模型评估：AUC 的陷阱

| 维度 | 说明 |
|:-----|:-----|
| **AUC 受什么影响** | ① Prevalence（物种出现率越低 → AUC 越高）② 背景点的选择 ③ 研究区范围 |
| **为什么不只看 AUC** | AUC > 0.9 的模型可能因为背景范围过大而虚高，而非模型真的好 |
| **推荐替代** | TSS (True Skill Statistic) + Boyce 指数 + 灵敏度/特异度 |
| **好模型的标准** | AUC > 0.7 + TSS > 0.4 + Boyce > 0.5 |

### E2 生态位等价性/相似性检验 (ecospat)

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 检验两个物种/种群的生态位是否显著不同；常用在入侵物种、物种分化研究中 |
| **不适用场景** | ① 存在点太少（< 20）② 环境空间定义不合适（网格分辨率/范围的选择敏感） |
| **数据最低要求** | 每个物种 ≥ 20 个存在点；推荐 ≥ 50 |
| **稳健性** | ⭐⭐⭐ 对环境空间的定义（PCA 前几轴的选择、网格分辨率）比较敏感 |
| **假设违反后果** | 环境空间定义不当 → 等价性检验的结论可能反转 |
| **替代方案** | nicheROVER（贝叶斯框架，更灵活）；ENMtools（经典工具箱） |

## F — 网络与食物网

### 共现网络

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 从群落丰度数据推断物种间的潜在关联模式 |
| **不适用场景** | ① 需要推断因果关系 ② 含大量稀有物种 ③ 期望直接解释为互作关系 |
| **数据最低要求** | 样方数 ≥ 30；物种数 ≥ 10；稀有物种应先过滤 |
| **稳健性** | ⭐⭐ 假阳性率高（尤其是 Spearman 相关）；稀疏数据更严重 |
| **假设违反后果** | 将 Spearman 相关为正的物种理解为"互惠"→ 通常错误！它们可能只是共享环境偏好 |
| **替代方案** | 成分数据 → SparCC / SPIEC-EASI；需要推断因果 → 结构方程模型 + 实验验证 |

#### 注意：共现 ≠ 互作

```
正共现的可能原因：
  1. 共享环境偏好（非互惠）
  2. 互惠关系（真实互作）
  3. 共享扩散过程
  4. 第三者效应（都受第三个物种影响）

负共现的可能原因：
  1. 竞争排斥
  2. 不同环境偏好
  3. 取样伪迹
```

## G — 时间序列与动态

### G1 ARIMA

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 平稳（或差分后平稳）的单变量时间序列预测 |
| **不适用场景** | ① 有结构性断点 ② 序列太短 (< 30) ③ 多变量系统 |
| **数据最低要求** | n ≥ 30（bare minimum）；n ≥ 50 较好；季节性 ARIMA 需要 ≥ 2 个周期 |
| **稳健性** | ⭐⭐⭐ 对平稳性假设敏感；ADF 检验对短序列效力不足 |
| **假设违反后果** | 非平稳 → 虚假回归 → 结论完全不可靠 |
| **替代方案** | 有断点 → 分段回归 + change point；多变量 → VAR / MARSS；灵活趋势 → GAM |

### G2 早期预警信号 (EWS)

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 检测生态系统是否接近临界转换点（基于临界减慢 CSD 理论） |
| **不适用场景** | ① 序列太短 (< 30) ② 系统不存在临界转换 ③ 多个外部驱动同时作用 |
| **数据最低要求** | n ≥ 30（bare minimum）；n ≥ 50 推荐；n ≥ 100 可靠 |
| **稳健性** | ⭐⭐ 假阳性率较高；多个 EWS 指标一致时才可靠（方差+自相关+偏度同时上升） |
| **假设违反后果** | 序列太短 → EWS 指标波动大 → 假阳性；外部强迫 → EWS 可能不出现 |
| **替代方案** | 空间 EWS（空间复制替代时间复制）；基于模型的临界点估计 (generic_ews) |

## H — 系统发育与功能性状

### H1 系统发育信号

#### Pagel's λ vs Blomberg's K

| 维度 | Pagel's λ | Blomberg's K |
|:-----|:----------|:-------------|
| **范围** | 0（无信号）到 1（BM模式） | 0（无信号）到 >1（比BM更强） |
| **K > 1 的含义** | N/A | 近缘种比布朗运动预期更相似（趋异不足） |
| **K < 1 的含义** | N/A | 近缘种比布朗运动预期更不相似（趋同） |
| **稳健性** | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| **报告建议** | **两者都报告** | 两者互补 |

#### 系统发育信号检验的注意事项

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 检验性状进化是否受系统发育约束 |
| **不适用场景** | 树的分支长度无意义（如基于分类学的超度量树） |
| **数据最低要求** | 物种数 ≥ 10（bare minimum）；≥ 30 推荐 |
| **稳健性** | ⭐⭐⭐ 对树拓扑的不确定性敏感；用多个树（后验分布）重复检验更稳健 |
| **假设违反后果** | 树严重错误 → 信号估计偏差不可预测 |
| **替代方案** | 多树 → 取平均；无可靠树 → 分类学作为系统发育的代理 |

### H2 PGLS

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 在回归分析中校正物种间的系统发育非独立性 |
| **不适用场景** | ① 无显著系统发育信号时（PGLS ≈ OLS）② 只有 1 个预测变量且物种很少 |
| **数据最低要求** | 物种数 ≥ 20；≥ 30 推荐 |
| **稳健性** | ⭐⭐⭐⭐ Pagel's λ 同时估计非常稳健；固定 λ 降低了灵活性 |
| **假设违反后果** | 不检验系统发育信号就做 PGLS → 可能在不必要时增加模型复杂度但无收益 |
| **替代方案** | 无系统发育信号 → 普通 OLS/GLM；有测量误差 → 系统发育混合模型 |

## I — 统计建模与机器学习

### I1 GLM

#### 过离散问题（计数 GLM）

| 维度 | 说明 |
|:-----|:-----|
| **什么是过离散** | 方差 > 均值（Poisson 假设方差=均值） |
| **如何检测** | `dispersion = residual deviance / residual df`；> 1.5 需换 NB |
| **稳健性** | Poisson GLM 对过离散**非常不稳健**：会严重低估标准误 → 假阳性 |
| **后果** | 过离散 + Poisson → 每个 p 值都不可靠（通常过于显著） |
| **解决方案** | ① 负二项 GLM (`glm.nb` / `glmmTMB`) ② quasi-Poisson ③ 零膨胀模型 |

#### GLM 的分布族选择

| 你的数据 | 第一选择 | 备选 |
|:---------|:---------|:-----|
| 连续/正态 | Gaussian | Gamma（偏态） |
| 计数（无过离散） | Poisson | — |
| 计数（过离散） | 负二项 (NB) | quasi-Poisson |
| 计数（零过多） | Zero-inflated NB / Hurdle | — |
| 二元 (0/1) | Binomial (logit) | Probit, cloglog |
| 比例 (0-1) | Beta回归 | quasibinomial |
| 正偏态 | Gamma (log link) | Inverse Gaussian |

---

### I2 混合效应模型 (LMM/GLMM)

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 嵌套/重复测量/分层数据，需要区分固定效应和随机效应 |
| **不适用场景** | ① 随机效应的组数 < 5（组间方差的估计极不稳定）② 只有 1 个分组因子且没有重复 |
| **数据最低要求** | 随机效应：≥ 5 组；≥ 10 组推荐；每组的观测：≥ 2 |
| **稳健性** | ⭐⭐⭐ 对小群组数敏感；组数 < 5 时随机效应方差估计可能为 0 或严重偏倚 |
| **假设违反后果** | 组数太少 → 方差估计为零或极端值 → 随机效应等同不存在 → 模型退化为 GLM |
| **替代方案** | 组数 < 5 → 改用固定效应（虽不优雅但更可靠）；贝叶斯 GLMM（先验可正则化） |

#### GLMM 的收敛问题

```
如果 GLMM 不收敛：
1. 检查 scale（变量标准化）
2. 更换优化器 (glmmTMB 比 lme4 更稳健)
3. 简化随机效应结构（先去掉随机斜率）
4. 换贝叶斯框架 (brms)
```

### I3 GAM (广义加性模型)

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 非线性关系未知时；比手动多项式回归更灵活 |
| **不适用场景** | ① 需要精确参数估计（GAM 给出的是平滑曲线，不是参数）② 外推预测 |
| **数据最低要求** | n ≥ 30（bare minimum）；每个平滑项：k ≤ n/3 |
| **稳健性** | ⭐⭐⭐⭐ 交叉验证自动选平滑度 (GCV/REML)；但外推极其危险 |
| **假设违反后果** | k 太小 → 过度平滑 → 漏掉真实模式；k 太大 → 过度拟合 → 虚假波动 |
| **替代方案** | 需要参数化 → 分段回归；需要外推 → 基于过程的模型 |

### I4 机器学习 vs 统计建模

| 维度 | 统计建模 (GLM/GAM) | 机器学习 (RF/GBM) |
|:-----|:--------------------|:--------------------|
| **主要目标** | 因果推断、假设检验 | 预测精度 |
| **可解释性** | ⭐⭐⭐⭐⭐ | ⭐⭐（RF有变量重要性，但无方向/显著性） |
| **外推能力** | ⭐⭐⭐⭐ （在模型假设范围内） | ⭐（基本上不能外推） |
| **小样本** | ⭐⭐⭐⭐ （n > 10k） | ⭐⭐ （n > 100 才比较可靠） |
| **交互作用** | 需要手动指定 | 自动捕获 |
| **非线性** | 需要手动建模 | 自动捕获 |
| **稳健性** | 对假设违反敏感 | 对分布假设非常稳健 |

## J — 模拟与随机化

### J1 零模型

#### 零模型选择的黄金法则

| 维度 | 说明 |
|:-----|:-----|
| **核心原则** | 零模型必须在"保留真实数据结构"和"打破待检验的生态过程"之间取得平衡 |
| **最容易犯的错** | 保留太多 → 过于保守 → 真效应被埋没；保留太少 → 过于宽松 → 假阳性泛滥 |
| **稳健性** | ⭐⭐⭐ 结论对零模型选择敏感——不同零模型可能得出完全相反的结论 |

#### 群落共现零模型对比

| 零模型 | 保留 | 破坏 | 保守程度 | 推荐场景 |
|:-------|:-----|:-----|:---------|:---------|
| quasiswap / curveball | 行和+列和 | 物种关联 | ⭐⭐⭐ 中等 | **推荐首选** |
| r00 | 行和+列和 | 关联 | ⭐⭐⭐ 中等 | 适合二分网络 |
| r1 | 行和 | 列和+关联 | ⭐⭐ 宽松 | 物种丰富度差异大时 |
| r0 | 列和 | 行和+关联 | ⭐⭐ 宽松 | 样方总丰度差异大时 |
| FF | 出现频率 | 物种关联 | ⭐ 最宽松 | 谨慎使用 |

#### 系统发育零模型

| 维度 | 说明 |
|:-----|:-----|
| **taxa.labels** | 重新随机分配物种名到树上 → 保留树拓扑，破坏性状-物种对应 |
| **richness** | 随机从整个物种池中抽取 → 保留物种多样性，破坏物种身份 |
| **结论敏感性** | 不同零模型给出不同的 ses.MPD/ses.MNTD 显著性，必须选择生态学上合理的 |

### J2 自举 (Bootstrap)

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | 估计参数的置信区间，尤其是参数分布未知时 |
| **不适用场景** | ① n < 15（bootstrap 分布离散）② 时间/空间自相关数据（需 block bootstrap） |
| **数据最低要求** | n ≥ 15（bare minimum）；n ≥ 30 推荐 |
| **稳健性** | ⭐⭐⭐⭐ n ≥ 30 时非常可靠；BCa 区间比百分位区间更准确 |
| **假设违反后果** | 小样本 → bootstrap 区间过窄（低估不确定性）；忽略自相关 → 更严重的区间偏倚 |
| **替代方案** | 小样本 → 贝叶斯（利用先验信息）；自相关 → block bootstrap / 时空 bootstrap |

### J3 贝叶斯方法

| 维度 | 说明 |
|:-----|:-----|
| **适用场景** | ① 有先验知识可利用 ② 复杂层次结构 ③ 小样本时的正则化 |
| **不适用场景** | ① 没有合理的先验 ② 受众不熟悉贝叶斯 ③ 模型太简单（GLM 够用了） |
| **稳健性** | ⭐⭐⭐ '客观'先验的选择需要理由；后验对先验的敏感性必须检验 |
| **假设违反后果** | 先验选择不当 → 后验偏倚（小样本时尤其明显）；未做先验敏感性分析 → 结论不可靠 |
| **替代方案** | 频率派方法（当先验难以选择时）；INLA（计算更快，但灵活度不如 MCMC） |

#### 先验敏感性分析

```
必须报告的贝叶斯诊断：
1. R-hat < 1.01（链收敛）
2. 有效样本量 > 1000
3. 先验 vs 后验分布图
4. 不同先验下的后验变化
5. LOO-CV / WAIC（模型比较）
```

## 📋 快速检查清单

当审稿人或合作者质疑你的方法时，用这个清单自检：

### 所有方法通用
- [ ] 是否检查了方法的假设前提？
- [ ] 数据量是否满足最低要求？
- [ ] 是否进行了必要的诊断检验？
- [ ] 方法的局限性是否在 Discussion 中讨论了？

### 群落数据专用
- [ ] 数据转换是否正确（Hellinger vs 无转换）？
- [ ] 梯度长度是否检查了（DCA 第一轴）？
- [ ] 双零问题是否处理了？
- [ ] betadisper 是否在 PERMANOVA 前做了？

### 建模专用
- [ ] 残差是否检查了（QQ图、残差vs拟合）？
- [ ] 过离散/零膨胀是否处理了？
- [ ] 共线性是否诊断了（VIF）？
- [ ] 模型选择是基于信息准则还是 p 值 mining？

### 空间/时间专用
- [ ] 空间自相关是否检验了？
- [ ] 时间自相关是否校正了？
- [ ] 是否忽略了重复测量设计？

---

> 💡 **核心原则**：方法的稳健性取决于你如何对待它的假设。了解边界比跑出 p < 0.05 更重要。
