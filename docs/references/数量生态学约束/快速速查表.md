## 🚨 十大"致命错误"（审稿人最常指出的问题）

| # | 错误 | 正确做法 | 类别 |
|:--|:-----|:---------|:----:|
| 1 | 群落数据直接跑 PCA（没用 Hellinger） | `decostand(spe, "hellinger")` → `rda()` | C |
| 2 | PERMANOVA 前不做 betadisper | `betadisper(dist, group)` → `anova()` | C |
| 3 | Poisson GLM 过离散不管（dispersion>1.5） | 换负二项：`glm.nb()` 或 `glmmTMB(...,family=nbinom2)` | I |
| 4 | 只看 AUC 评估 SDM | + TSS + Boyce 指数 | E |
| 5 | Shannon vs Simpson 只报告一个 | **两者都报告** + Hill q1, q2 | B |
| 6 | 组数 <5 却用 GLMM 的随机效应 | 改用固定效应 | I |
| 7 | maxnet 用默认参数不调参 | ENMeval 调 features + regularization | E |
| 8 | 零模型只跑一种 | 至少比较 2 种（如 quasiswap + taxa.labels） | J |
| 9 | Box-Cox 自动选 λ 前不处理异常值 | 先 winsorize → 再 Box-Cox | A |
| 10 | 混用 Spearman 相关和共现网络说"互作" | 在论文中声明"共现≠互作" | F |

## 🔢 核心数值阈值速查

### 数据量要求

| 方法 | 最小 | 推荐 | 备注 |
|:-----|:----:|:----:|:-----|
| GLM 每参数 | 10 观测 | 20 观测 | — |
| GLMM 组数 | 5 组 | 10 组 | <5 → 固定效应 |
| MEM/PCNM | 20 样方 | 50 样方 | — |
| MaxEnt 存在点 | 15 点 | 50 点 | <15 不稳定 |
| Chao1 (F1,F2) | F1>0, F2>0 | F1>5, F2>5 | F2=0→不可用 |
| ARIMA | 30 时间点 | 50 时间点 | — |
| EWS | 30 时间点 | 50 时间点 | — |
| Bootstrap | 15 观测 | 30 观测 | <15→贝叶斯 |
| Moran's I | 30 样方 | 50 样方 | — |

### 诊断阈值

| 指标 | 阈值 | 动作 |
|:-----|:-----|:-----|
| DCA 第一轴 SD | <3 → 线性 / 3-4 → 过渡 / >4 → 单峰 | 决定 RDA vs CCA |
| NMDS Stress | <0.1 好 / 0.1-0.2 OK / >0.2 差 | >0.2 换方法 |
| VIF 共线性 | >10 | 删变量 |
| 过离散 (dispersion) | >1.5 | 换负二项 |
| AUC | >0.7 OK / >0.9 好 | 同时看 TSS |
| TSS | >0.4 OK / >0.6 好 | 不要只看 AUC |
| |r| (共线性) | >0.7 | 删一个变量 |
| Cook's D | >4/n | 检查该点 |
| IndVal | >0.7 严格 / >0.5 探索 | — |
| Shannon p (正态性) | >0.05 → 正态 | n>5000 忽略 |

## 🔄 核心诊断流程

### 群落数据分析标准链路

```
原始数据
  → A3 Hellinger 转换（⭐必做）
  → DCA 梯度长度检查
    → <3 SD → RDA → 变量选择(VIF<10) → 方差分解(VPA)
    → >4 SD → CCA
  → 可视化：NMDS (Bray-Curtis) + envfit
  → 检验：PERMANOVA（⭐betadisper 前置！）
  → 解释：SIMPER / IndVal
  → 空间：MEM + 方差分解（环境 vs 空间）
```

### 统计建模标准链路

```
响应变量类型检查
  → 计数 → 检查过离散(dispersion>1.5? → NB)
            → 检查零膨胀(0%>30%? → ZI/Hurdle)
  → 连续 → 检查正态性(|skew|<1?)
  → 二元 → Logistic (binomial GLM)
  → 比例 → Beta 回归
  → 多变量 → mvabund

设计检查
  → 独立 → GLM/GAM
  → 嵌套 → GLMM (组数≥5?)
  → 重复测量 → GLMM + corAR1 / GAMM
  → 空间 → 空间 GLMM / 空间 GAM

模型后检查
  → QQ 图 + 残差 vs 拟合
  → 残差空间自相关? (Moran's I on residuals)
  → 残差时间自相关? (ACF)
```

## 🧭 方法选择快速导航

```
数据长什么样？                         → 去哪里？
物种-样方矩阵                          → B 多样性 / C 多元统计
物种+环境变量                          → C RDA/CCA / E SDM
物种+空间坐标                          → D 空间生态学
物种+时间序列                          → G 时间序列
单变量响应                             → I GLM/GAM
物种间互作                             → F 网络
系统发育树+性状                        → H 系统发育

问题是什么？                           → 用什么方法？
"群落组成多样吗？"                      → B1 α多样性
"组间有显著差异吗？"                    → C3 PERMANOVA
"什么环境驱动了群落变化？"              → C1 RDA/CCA + VPA
"空间结构重要吗？"                      → D2 MEM + VPA
"物种分布在哪里？"                      → E2 MaxEnt / biomod2
"系统有什么趋势？"                      → G2 ARIMA / GAM
"性状受进化约束吗？"                    → H1 系统发育信号 + H2 PGLS
"怎么建模和预测？"                      → I1-I6 (看数据类型)
"模式是随机的吗？"                      → J1 零模型
```

## 📦 应急 R 代码片段

```r
# 快速 Hellinger + PCA
spe_hel <- decostand(spe, "hellinger"); rda(spe_hel)

# 快速 NMDS + 检验
nmds <- metaMDS(vegdist(spe, "bray"), try=100); stressplot(nmds)

# 快速 PERMANOVA 链路
dist <- vegdist(spe, "bray")
bd <- betadisper(dist, group); anova(bd)      # 先同质性
adonis2(dist ~ group, permutations=9999)       # 再检验

# 快速 GLM 过离散
mod <- glm(y ~ x, family=poisson)
disp <- sum(resid(mod, type="pearson")^2)/df.residual(mod)
if(disp > 1.5) glm.nb(y ~ x)  # 换负二项

# 快速 VIF
vif.cca(rda(spe_hel ~ ., data=env))

# 快速系统发育信号
phylosig(tree, trait, method="lambda", test=TRUE)

# 快速 iNEXT 稀疏化
library(iNEXT); iNEXT(spe, q=c(0,1,2), datatype="abundance")
```
