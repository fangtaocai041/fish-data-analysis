# 🌲 方法选择决策树

> **根据你的数据类型和科学问题，找到最合适的分析方法**
>
> 从数据特征出发 → 逐步筛选 → 定位到具体分类和方法

---

## 第一步：你的数据长什么样？

```
你的数据类型？
│
├─ 物种-样方矩阵（群落数据）
│   └─ → B-多样性分析 / C-多元统计分析
│
├─ 物种+环境变量
│   └─ → C-多元统计分析（约束排序）/ E-生态位与物种分布
│
├─ 物种+空间坐标
│   └─ → D-空间生态学
│
├─ 物种+时间序列
│   └─ → G-时间序列与动态
│
├─ 单变量响应（如体长、丰度）
│   └─ → I-统计建模
│
├─ 物种间相互作用数据
│   └─ → F-网络与食物网
│
├─ 系统发育树+性状数据
│   └─ → H-系统发育与功能性状
│
└─ 不确定 → 看第二步
```

## 第二步：你的科学问题是什么？

```
你的科学问题？
│
├─ "群落的物种组成有多样？"
│   └─ 多样性指数 → B1 α多样性
│       └─ 群落间差异 → B2 β多样性
│
├─ "群落组成在组间是否有显著差异？"
│   └─ 单一时间点 → PERMANOVA (C3)
│       └─ 多个时间点 → 重复测量PERMANOVA (C3 + G1)
│
├─ "哪些环境因子驱动了群落结构变化？"
│   └─ 筛选关键因子 → RDA/CCA (C1)
│       └─ 方差分解 → VPA (C1)
│
├─ "物种分布受什么因素控制？"
│   └─ 有存在/不存在数据 → MaxEnt/GLM (E2/I1)
│       └─ 只有存在数据 → MaxEnt/ENFA (E2/E3)
│
├─ "空间距离是否影响群落相似性？"
│   └─ 距离衰减关系 → D1 空间自相关
│       └─ 空间结构分解 → MEM/PCNM (D3)
│
├─ "生态过程是否存在时间趋势？"
│   └─ 单个时间序列 → G1 分解+G3 变化点
│       └─ 多个时间序列 → G2 状态空间模型
│
├─ "物种之间的关联模式是什么？"
│   └─ 物种共现 → F2 共现网络
│       └─ 营养关系 → F1 食物网
│
├─ "性状是否受系统发育约束？"
│   └─ 计算信号 → H1 系统发育信号
│       └─ 校正系统发育 → PGLS (H2)
│
├─ "观测到的模式是否显著？"
│   └─ 零模型检验 → J1 零模型
│       └─ 参数不确定性 → J2 置换检验
│
└─ "如何预测？"
    └─ 解释性 → GLM/GLMM/GAM/SEM (I1-I4)
        └─ 预测性 → 随机森林/GBM/深度学习 (I5-I6)
```

## 第三步：你的数据特征是什么？

```
数据类型 × 分布特征 → 方法选择修正
│
├─ 连续数据（正态）
│   └─ 参数方法适用 (I1 GLM高斯族)
│
├─ 连续数据（偏态）
│   └─ 转换 → 非参数 / 广义模型 (I1 GLM伽马族/逆高斯族)
│
├─ 计数数据 / 丰度数据
│   └─ Poisson / 负二项 GLM (I1)
│       └─ 零膨胀 → zero-inflated / hurdle 模型 (I2)
│
├─ 二元数据（存在/不存在）
│   └─ Logistic 回归 / MaxEnt (I1/E3)
│
├─ 比例数据（0-1）
│   └─ Beta 回归 / Logistic 回归 (I1)
│
├─ 等级/有序数据
│   └─ 累积链接模型 / 非参数检验 (I1)
│
└─ 多元响应（多个物种同时）
    └─ 多元GLM → mvabund (I1变异)
```

## 第四步：你的实验/抽样设计？

```
设计类型 → 模型结构
│
├─ 独立样本
│   └─ 标准方法可用
│
├─ 嵌套/分层抽样
│   └─ + 随机效应 → GLMM / LMM (I2)
│
├─ 时间重复测量
│   └─ + 时间自相关 → GAM / GLMM + corAR1 (I3/I2)
│
├─ 空间自相关
│   └─ + 空间效应 → 空间GLM / 空间GAM (D1 + I1/I3)
│
└─ 系统发育相关
    └─ + 系统发育校正 → PGLS (H2)
```
