## 🎯 方法适用边界与稳健性

每种方法都有它的边界。**知道什么情况下不能用，比知道怎么用更重要。**

👉 **`_方法适用边界/适用边界与稳健性全面指南.md`**

涵盖全部 A-J 十类方法：
- **适用场景** vs **不适用场景** — 明确边界
- **数据最低要求** — 样本量、变量数、组数
- **稳健性评级** (⭐~⭐⭐⭐⭐⭐) — 对标假设违反的敏感度
- **假设违反后果** — 如果犯规会怎样
- **替代方案** — 当前方法不可用时的 Plan B

关键诊断速查：
- GLM 过离散 → 负二项 | PERMANOVA 异质性 → betadisper
- Mantel 检验效力低 → MEM+RDA | MaxEnt 采样偏差 → 空间校正
- AUC 虚高 → TSS+Boyce | NMDS Stress 高 → 换距离或维度
