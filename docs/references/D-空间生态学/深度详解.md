# 🗺️ D — 空间生态学 · 深度详解

> 生态数据几乎总是存在空间结构。忽视它 → 假阳性泛滥；正确建模 → 揭示扩散、环境筛选等关键过程。

---

## D1 — 空间自相关：检测、理解和应对

### 一、为什么空间自相关是"双刃剑"？

```
好消息：空间自相关本身是生态信号
  → "近处更相似"说明扩散限制或环境梯度在起作用
  → 空间结构可以替代"未测量的环境变量"

坏消息：空间自相关破坏大多数统计方法的独立性假设
  → 虚假回归（看起来显著，实际只是空间趋势）
  → 过度自信的 p 值（标准误被低估）
  → 自由度虚高（n=100 个样方 ≠ 100 个独立信息）
```

### 二、Moran's I 的完整解读

```
Moran's I 的范围：-1（完美分散）到 +1（完美聚集），期望值 ≈ -1/(n-1)

正值（聚集）：相邻样方相似 → 环境梯度或扩散限制
零值（随机）：样方在空间上独立
负值（分散）：相邻样方不同 → 竞争排斥或生境斑块化

关键陷阱：
1. Moran's I 只是"全局"指标 → 可能掩盖局部异质性
2. 结果对空间权重矩阵的定义极其敏感
3. 需要置换检验（而非正态近似）来获得可靠 p 值
```

### 三、空间权重矩阵：最关键的决策

```
邻居定义决定了 Moran's I 的结果：

方法1: 距离阈值 (Distance-based)
  → 所有距离 < d 的点互为邻居
  → 选择 d = 最大最小生成树的最长边（常用）
  → 问题：采样密度不均时，密集区邻居过多

方法2: KNN (K-Nearest Neighbors)
  → 每个点选最近的 K 个点为邻居
  → K 的选择：K=8 是经验默认
  → 问题：孤立的点也被迫有 K 个邻居

方法3: Delaunay 三角剖分
  → 点之间的自然邻接关系
  → 适合不规则采样（如沿河流）

推荐做法：
→ 至少比较 2-3 种权重矩阵
→ 如果 Moran's I 的符号和显著性不随权重定义改变 → 稳健
→ 如果不一致 → 在论文中需要讨论
```

### 四、Mantel 检验：为什么不推荐？

```
Mantel 检验在生态学中曾被广泛使用，但近十年被系统性地批判：

三个致命问题：
1. 统计效力极低 — 即使真实的距离衰减存在，也可能不显著
2. 对非线性不敏感 — 如果距离衰减只在特定尺度发生，Mantel 可能漏掉
3. 不能区分"环境效应"和"空间效应"

替代方案（按推荐度排序）：
1. ✅ MEM + RDA 方差分解 — 最推荐
2. ✅ db-RDA + 空间变量 — 直接建模
3. ⚠️ Procrustes 检验 — 特定场景
4. ❌ Mantel 检验 — 除非审稿人坚持要求
```

## D2 — MEM / PCNM：空间特征变量法

### 一、核心思想（简洁版）

```
MEM (Moran's Eigenvector Maps) = 从空间坐标中提取"正交的空间模式"

每个 MEM 特征向量 = 一个特定空间尺度的模式
  正特征值 → 正空间自相关（近处相似）← 生态学最常用
  负特征值 → 负空间自相关（近处不同）

低序号 MEM → 大尺度空间结构（如：上游 vs 下游）
高序号 MEM → 小尺度空间结构（如：微生境斑块）
```

### 二、MEM 的使用场景

```
场景1: 方差分解 (VPA)
  spe_hel ~ env_vars + MEM_vars
  → 分解为：环境解释 + 空间解释 + 共享 + 残差
  → 回答："群落变异中有多少是环境 vs 空间？"

场景2: 空间校正
  spe_hel ~ env_vars + MEM1  # MEM1 作为"空间协变量"
  → 在校正了空间结构后，环境变量的效应还显著吗？

场景3: 多尺度分析
  分别用不同尺度的 MEM 建模
  → 哪些生态过程在大尺度运作？哪些在小尺度？
```

### 三、MEM 的正特征值选择

```
dbmem() 默认只保留正特征值 → 对应"正空间自相关"

为什么只保留正特征值？
→ 负特征值代表"近处更不同"（负自相关）
→ 在大多数生态系统中，正自相关是主导的
→ 负自相关通常来自竞争排斥，且统计上较弱

但！如果生态假说是"竞争导致排斥" → 可能需要负特征值
→ 使用 MEM.autocor = "all" 保留全部
```

```r
# ============================================================
# D 增强版：空间分析完整诊断
# ============================================================
library(spdep)       # 空间自相关 + 权重矩阵
library(adespatial)  # MEM / dbMEM
library(vegan)

# ---- 1. 空间权重矩阵选择 ----
# 比较三种权重定义
compare_weights <- function(coords) {
  # Delaunay 三角剖分
  nb_tri <- tri2nb(coords)
  
  # KNN (k=8)
  nb_knn <- knn2nb(knearneigh(as.matrix(coords), k = 8))
  
  # 距离阈值（最大最小生成树的最长边）
  thresh <- max(unlist(nbdists(nb_tri, coords)))
  nb_dist <- dnearneigh(as.matrix(coords), 0, thresh)
  
  cat("=== 权重矩阵比较 ===\n")
  cat("Delaunay:  平均邻居数 =", 
      round(mean(card(nb_tri)), 1), "\n")
  cat("KNN(k=8):  邻居数 = 8 (固定)\n")
  cat("距离阈值:  平均邻居数 =",
      round(mean(card(nb_dist)), 1), "\n")
  
  return(list(delaunay = nb_tri, knn = nb_knn, threshold = nb_dist))
}

# ---- 2. 多尺度 Moran's I ----
# 使用 sp.correlogram 检查自相关随距离的变化
multi_scale_moran <- function(x, coords, n_classes = 10) {
  nb <- knn2nb(knearneigh(as.matrix(coords), k = 8))
  listw <- nb2listw(nb, style = "W")
  
  # 空间自相关图谱
  cor_sp <- sp.correlogram(nb, x, order = n_classes, 
                            method = "I", zero.policy = TRUE)
  plot(cor_sp, main = "Moran's I 随距离的变化")
  
  # 解读：
  # I 在某距离后降到 ~0 → 自相关的特征尺度
  # I 始终为正 → 空间结构在所有尺度存在
  return(cor_sp)
}

# ---- 3. MEM 构建与诊断 ----
build_mem_diagnostic <- function(coords, spe_hel) {
  # 构建 MEM
  thresh <- give.thresh(dist(coords))
  mem_all <- dbmem(coords, thresh = thresh[1],
                   MEM.autocor = "positive")
  
  # 提取显著 MEM
  rda_mem <- rda(spe_hel ~ ., data = as.data.frame(mem_all))
  sel <- ordistep(rda(spe_hel ~ 1, data = as.data.frame(mem_all)),
                  scope = formula(rda_mem),
                  direction = "forward", pstep = 999)
  
  cat("\n=== MEM 诊断 ===\n")
  cat("总 MEM 数:", ncol(mem_all), "\n")
  cat("显著 MEM 数:", length(attr(sel$terms, "term.labels")), "\n")
  
  # MEM 空间可视化（选前 3 个正特征向量）
  op <- par(mfrow = c(1, 3))
  for (i in 1:min(3, ncol(mem_all))) {
    # 点的大小表示 MEM 的值
    plot(coords, cex = abs(mem_all[, i]) * 3,
         pch = ifelse(mem_all[, i] > 0, 16, 1),
         col = ifelse(mem_all[, i] > 0, "red", "blue"),
         main = paste0("MEM", i))
  }
  par(op)
  
  return(list(mem = mem_all, selected = sel))
}

# ---- 4. 空间自相关校正前后对比 ----
# 模型残差的空间自相关检查
check_residual_autocorr <- function(model, coords) {
  resid <- residuals(model)
  nb <- knn2nb(knearneigh(as.matrix(coords), k = 8))
  listw <- nb2listw(nb, style = "W")
  
  mi <- moran.test(resid, listw)
  cat("\n=== 残差空间自相关 ===\n")
  cat("Moran's I =", round(mi$estimate[1], 3), 
      "p =", round(mi$p.value, 4), "\n")
  
  if (mi$p.value < 0.05) {
    cat("⚠️ 残差仍有显著空间自相关 → 模型未充分校正空间结构\n")
    cat("建议：增加更多的 MEM 作为协变量\n")
  } else {
    cat("✅ 残差无显著空间自相关\n")
  }
}
```
