## 一、共现网络：最常见也最易被误读

### "正相关 ≠ 互惠" → 四种可能原因

```
物种A和物种B正共现的可能原因：
1. 🟢 共享环境偏好（最常见，不是互作！）
   → 都喜欢高氧环境 → 在高氧样方一起出现

2. 🟢 真实互惠关系
   → 如固氮菌 + 植物

3. 🟡 共享扩散途径
   → 都在洪水期被冲到下游

4. 🟡 第三者效应
   → 物种C抑制了A和B的竞争者 → A和B间接正相关

只凭群落丰度数据 → 你无法区分这四种！
→ 需要实验验证或至少机制推理
```

### 网络构建方法选择

| 方法 | 适用 | 假阳性控制 | 假阴性 |
|:-----|:-----|:----------|:------|
| **Spearman 相关** | 任意 | ❌ 差 | 中等 |
| **SparCC** | 成分数据 | ⭐⭐⭐ | 中等 |
| **SPIEC-EASI** | 高维成分数据 | ⭐⭐⭐⭐ | 较高 |
| **贝叶斯 (ebcsp)** | 需要不确定度量 | ⭐⭐⭐ | 中等 |

### 共现网络完整代码

```r
library(igraph)
library(Hmisc)
library(vegan)

# 从群落数据构建共现网络
# spe: 样方×物种矩阵（稀有物种已过滤）

# 计算 Spearman 相关 + 显著性
cor_test <- rcorr(as.matrix(spe), type = "spearman")
cor_mat <- cor_test$r
p_mat   <- cor_test$P

# 多重比较校正（Bonferroni）
n_tests <- ncol(spe) * (ncol(spe) - 1) / 2
p_mat_adj <- p.adjust(p_mat, method = "bonferroni", n = n_tests)
p_mat_adj <- matrix(p_mat_adj, nrow = nrow(p_mat))

# 只保留显著 + 强相关
cor_mat[p_mat > 0.05] <- 0        # 不显著 → 0
cor_mat[abs(cor_mat) < 0.3] <- 0  # 弱相关 → 0
diag(cor_mat) <- 0               # 自相关 → 0

# 构建 igraph
g <- graph_from_adjacency_matrix(cor_mat, 
  mode = "undirected", weighted = TRUE, diag = FALSE)

# 正负边分开着色
E(g)$color <- ifelse(E(g)$weight > 0, "#4575B4", "#D73027")
E(g)$width <- abs(E(g)$weight) * 3

# 可视化
set.seed(42)
plot(g, vertex.size = log(colSums(spe)) * 3,
     vertex.label.cex = 0.6,
     vertex.color = "lightblue",
     main = "物种共现网络 (Spearman |r|>0.3, p<0.05)",
     sub = "蓝色=正相关  红色=负相关  线宽=相关强度")
```

## 二、二分网络：嵌套性和模块性

### 嵌套性 vs 模块性：两种"有序结构"

```
嵌套 (Nestedness):
→ 特化种互作是泛化种互作的子集
→ 例：岛屿上特化传粉者访问的植物是泛化者的子集
→ 指标：NODF, WNODF

模块性 (Modularity):
→ 物种形成紧密的"模块"，模块内连接多，模块间连接少
→ 例：花管长度匹配特定蜂类口器 → 形成模块
→ 指标：Q (Barber's Q for bipartite)

两种有序结构可以共存于同一网络！
不需要二选一，两者都报告。

### 二分网络完整代码

```r
library(bipartite)

# bip_mat: 行=植物 列=传粉者（或任意两组物种的互作矩阵）

# ---- 网络级别指标 ----
metrics <- networklevel(bip_mat,
  index = c("connectance", "web asymmetry", 
            "links per species", "nestedness",
            "NODF", "weighted NODF", "H2",
            "number of compartments"))
print(metrics)

# ---- 嵌套性检验 ----
nodf_obs <- nested(bip_mat, method = "NODF")

# 零模型：随机化网络结构 999 次
nulls <- nullmodel(bip_mat, N = 999, method = "r2d")
null_nodf <- sapply(nulls, function(x) {
  tryCatch(nested(x, method = "NODF"), error = function(e) NA)
})
p_nodf <- (sum(null_nodf >= nodf_obs, na.rm = TRUE) + 1) / 
          (sum(!is.na(null_nodf)) + 1)
cat(sprintf("NODF = %.2f, 零模型 p = %.3f\n", nodf_obs, p_nodf))

# ---- 模块性分析 ----
modules <- computeModules(bip_mat, method = "Beckett")
plotModuleWeb(modules)  # 可视化模块

# ---- 物种级别特化度 d' ----
species_spec <- specieslevel(bip_mat, index = "d")
# d'高 → 特化种；d'低 → 泛化种
head(species_spec$`higher level`)  # 高级消费者
head(species_spec$`lower level`)   # 低级资源
```

### 食物网完整代码

```r
library(igraph)

# mat: 捕食矩阵（行=捕食者 列=猎物，1/0）
g <- graph_from_adjacency_matrix(mat, mode = "directed")

# ---- 基础拓扑指标 ----
S <- vcount(g)              # 物种数
L <- ecount(g)              # 连接数
C <- L / (S * (S - 1))      # Connectance (有向图)
avg_path <- mean_distance(g) # 平均食物链长度
max_path <- diameter(g)      # 最长食物链

# ---- 节点重要性 ----
in_deg  <- degree(g, mode = "in")   # 猎物数（入度）
out_deg <- degree(g, mode = "out") # 天敌数（出度）
btw <- betweenness(g)               # 介数中心性——关键物种

# 找出"keystone"物种（介数最高的前5个）
keystone <- names(sort(btw, decreasing = TRUE)[1:5])
cat("潜在关键物种:", paste(keystone, collapse = ", "), "\n")

# ---- 模块性 ----
comm <- cluster_fast_greedy(as.undirected(g))
Q <- modularity(comm)
cat(sprintf("模块性 Q = %.3f\n", Q))

# ---- 可视化 ----
plot(g,
     vertex.size = log(in_deg + 1) * 4,
     vertex.color = ifelse(in_deg > out_deg, "#D73027", "#4575B4"),
     edge.arrow.size = 0.3,
     vertex.label.cex = 0.7,
     main = "食物网 (红=主要作为猎物  蓝=主要作为捕食者)")
```

## 三、网络比较的核心陷阱

```
❌ 错误：比较两个网络的 Connectance，说"网络A更连接"
→ Connectance 随网络大小增加而下降
→ 除非两个网络节点数相同，否则不能直接比较！

✅ 正确：用零模型标准化
→ 构建等大小随机网络 → 比较观测值 vs 零模型的 z-score
→ 报告："网络A的连接度显著高于同大小随机网络"
```
